====================================
    2FA認証マネージャー v1.0
====================================

【実行方法】
flutter_2fa_manager.exe をダブルクリックして実行してください。

【重要な注意事項】

🚨 Windows Defenderの警告について：
  - 初回実行時に「WindowsによってPCが保護されました」という警告が表示される場合があります
  - これは署名されていない実行ファイルに対する標準的な警告です
  - 対処方法：
    1. 「詳細情報」をクリック
    2. 「実行」ボタンをクリック
    3. アプリケーションが起動します

📁 ファイル構成：
  - flutter_2fa_manager.exe ← メインプログラム
  - flutter_windows.dll     ← 必須ライブラリ（削除禁止）
  - data/                   ← アプリケーションデータ（削除禁止）

⚠️  注意：
  - すべてのファイルを同じフォルダに保持してください
  - exe単体では動作しません
  - ウイルス対策ソフトが反応する場合は、フォルダを除外設定に追加してください

【アンインストール方法】
  このフォルダを削除するだけでアンインストール完了です。
  レジストリへの書き込みはありません。

【データの保存場所】
  設定とデータは以下に保存されます：
  %APPDATA%\flutter_2fa_manager\
