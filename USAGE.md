# 2FA管理器使用指南

## 快速开始

1. **启动应用**
   ```bash
   cd flutter_2fa_manager
   flutter run -d windows
   ```

2. **添加第一个2FA认证**
   - 点击右下角的"+"按钮
   - 输入账户信息和密钥
   - 点击"添加"保存

3. **获取验证码**
   - 点击任意2FA卡片
   - 验证码会显示并自动复制到剪贴板
